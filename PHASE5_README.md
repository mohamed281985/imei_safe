## 🚀 المهام الأربع الرئيسية

### ✅ Task 1: إصلاح Render (30-45 دقيقة)
```
المشكلة: Cannot find module 'csurf'
الحل: أضف csurf إلى dependencies وحدّث build script
الاختبار: curl /api/csrf-token يرجع token
```

### ✅ Task 2: تدوير المفاتيح (45-60 دقيقة)
```
1. مفتاح Firebase API (واجهة المستخدم)
2. Firebase Service Account (الخادم)
3. مفتاح Paymob API (المدفوعات)
4. Paymob Secret Key (المدفوعات)
5. مفاتيح Supabase (قاعدة البيانات)
6. Session Secret (المصادقة)
```

### ✅ Task 3: تفعيل RLS (30-45 دقيقة)
```
تفعيل Row Level Security على 5 جداول:
- users
- payments
- imei_searches
- notifications
- audit_logs

النتيجة: كل مستخدم يرى فقط بيانته
```

### ✅ Task 4: الفحص النهائي (30-45 دقيقة)
```
التحقق من 31 نقطة أمان:
✅ CSRF محمي
✅ الجلسات آمنة
✅ معدل الطلبات محدود
✅ جميع المتطلبات الأخرى مُوفرة
```
